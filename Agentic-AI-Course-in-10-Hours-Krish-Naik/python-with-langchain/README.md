# python-with-langchain Setup & Run

## Selecting Jupyter Kernel

### Option 1: In VS Code (Recommended)
1. Open the [1-langchain-intro.ipynb](langchain/1-langchain-intro.ipynb) file.
2. Look at the top-right corner of the notebook editor window. You will see a button indicating the current kernel (e.g., `Python 3.x.x` or `Select Kernel`).
3. Click on it and select **Select Kernel** (or click the active kernel name).
4. Select **Python Environments...** from the dropdown.
5. Choose the interpreter pointing to the local virtual environment: `./.venv/Scripts/python.exe` (often labeled as `python-with-langchain (.venv)`).
6. Run the notebook cell again.
